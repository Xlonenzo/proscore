# PASSA - Marketplace de Serviços Residenciais

## Stack
- Backend: FastAPI + SQLAlchemy + PostgreSQL (Render) / SQLite (local)
- Frontend: Jinja2 templates + vanilla CSS/JS
- Auth: JWT (python-jose) + bcrypt
- AI: Groq Vision API para precificação
- Payments: Stripe Connect (split)
- Deploy: Render (render.yaml)
